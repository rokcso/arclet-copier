**Arclet Copier - Smart URL Copy & Share**

## Summary (132 characters max)
Copy web links with batch copy, parameter filtering, Markdown format, short URLs & QR codes. Keyboard shortcuts, dark theme.

## Detailed Description

Arclet Copier is a powerful Chrome extension for quickly copying URLs with advanced features. Whether you need a clean URL, a Markdown link, or a short URL, Arclet Copier has you covered with its intelligent URL processing and customizable templates.

### üåü Key Features

**Smart URL Cleaning**
Three cleaning modes to handle URL parameters:
- Off: Keep all parameters intact
- Smart: Remove tracking parameters (utm_*, fbclid, etc.) while keeping functional ones
- Aggressive: Remove all parameters for the cleanest URL
- Custom parameter rules: Define your own tracking and functional parameters

**Custom Template System**
Create personalized copy formats with 11 template variables:
- {{url}}, {{originalUrl}}, {{title}}, {{hostname}}, {{domain}}
- {{shortUrl}}, {{date}}, {{time}}, {{datetime}}, {{timestamp}}, {{iso}}
- {{author}}, {{description}}
- Real-time preview and template validation
- Emoji icons for easy template identification

**Short URL Generation**
Generate short links with one click:
- Support for is.gd and TinyURL services
- Smart caching to avoid duplicate requests
- Persistent cache across browser sessions
- Batch short URL generation

**Batch Copy**
Copy multiple tab URLs at once:
- Filter by window scope (current/all windows)
- Web pages only filter to exclude system pages
- Duplicate URL removal
- Preview before copying
- Support for all copy formats and custom templates

**QR Code Generation**
Create QR codes for current page URLs:
- Copy QR code image to clipboard
- Theme-adaptive QR codes (light/dark mode)
- Clean interface with URL preview

**Rich Copy Formats**
- Plain URL
- Markdown format: [Page Title](URL)
- Short URL
- Custom templates with unlimited possibilities

**Multiple Ways to Copy**
- Click extension icon for popup interface
- Keyboard shortcut: Ctrl+Shift+C (Windows/Linux) or Cmd+Shift+C (Mac)
- Right-click context menu: "Copy Current URL"
- All methods respect your URL cleaning and format settings

**Beautiful Themes**
- 3 appearance modes: System, Light, Dark
- 5 theme colors: Orange, Yellow, Green, Blue, Purple
- Consistent theming across all UI elements
