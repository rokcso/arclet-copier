## 🛠️核心功能

📋多种复制格式
- 普通 URL - 原始链接或清理后的干净链接
- Markdown 格式 - [页面标题](URL) 格式，完美适配文档
- 自定义模板 - 使用变量创建专属格式模板
- 短链服务 - 支持 is.gd 和 TinyURL，智能缓存避免重复请求

⚡灵活的操作方式
- 扩展弹窗 - 点击图标打开完整功能界面
- 快捷键 - 可自定义
- 右键菜单 - 右键即可快速复制当前页面 URL
- 批量处理 - 一次性复制多个标签页链接
