# Chrome 网上应用店商品详情 - 中文

## 产品名称
**Arclet Copier - 智能URL复制与分享**

## 摘要（132字符以内）
复制网页链接，支持批量复制、参数过滤、Markdown 格式、短链和二维码生成。快捷键、深色主题。

## 详细说明

Arclet Copier 是一款功能强大的 Chrome 扩展程序，专为快速复制 URL 并提供高级特性而设计。无论您需要干净的 URL、Markdown 链接还是短链接，Arclet Copier 都能通过智能 URL 处理和可自定义模板满足您的需求。

### 🌟 核心功能

**智能URL清理**
三种清理模式处理URL参数:
- 关闭: 保留所有参数
- 智能清理: 移除跟踪参数(utm_*、fbclid等)，保留功能参数
- 激进模式: 移除所有参数，获得最干净的URL
- 自定义参数规则: 自定义您的跟踪参数和功能参数列表

**自定义模板系统**
使用11种模板变量创建个性化复制格式:
- {{url}}, {{originalUrl}}, {{title}}, {{hostname}}, {{domain}}
- {{shortUrl}}, {{date}}, {{time}}, {{datetime}}, {{timestamp}}, {{iso}}
- {{author}}, {{description}}
- 实时预览和模板验证
- Emoji图标便于识别模板

**短链生成**
一键生成短链接:
- 支持 is.gd 和 TinyURL 服务
- 智能缓存避免重复请求
- 跨浏览器会话持久化缓存
- 批量短链生成

**批量复制**
一次复制多个标签页URL:
- 按窗口范围筛选(当前窗口/所有窗口)
- 仅Web页面过滤，排除系统页面
- 自动去除重复URL
- 复制前预览
- 支持所有复制格式和自定义模板

**二维码生成**
为当前页面URL创建二维码:
- 复制二维码图片到剪贴板
- 主题自适应二维码(亮色/暗色模式)
- 简洁界面，带URL预览

**丰富的复制格式**
- 纯URL
- Markdown格式: [页面标题](URL)
- 短链接
- 自定义模板，实现无限可能

**多种复制方式**
- 点击扩展图标打开弹窗界面
- 键盘快捷键: Ctrl+Shift+C (Windows/Linux) 或 Cmd+Shift+C (Mac)
- 右键菜单: "复制当前 URL"
- 所有方式都遵循您的URL清理和格式设置

**精美主题**
- 3种外观模式: 跟随系统、浅色、深色
- 5种主题色: 橙色、黄色、绿色、蓝色、紫色
- 所有UI元素保持主题一致性

**多语言支持**
完整界面支持9种语言:
- 简体中文
- 繁體中文
- English (英语)
- Español (西班牙语)
- 日本語 (日语)
- Deutsch (德语)
- Français (法语)
- Português (葡萄牙语)
- Русский (俄语)
- 한국어 (韩语)

### 🎯 适用场景

- 开发者在文档或Bug报告中分享链接
- 内容创作者构建参考列表
- 研究人员管理网页来源
- 社交媒体管理者分发干净链接
- 任何需要高效复制和分享URL的人

### 🔒 隐私与权限

Arclet Copier 尊重您的隐私:
- 所有数据本地存储在您的设备上
- 不收集或跟踪数据
- 无外部服务器(可选的短链服务除外)
- 开源透明

所需权限说明:
- activeTab: 获取当前标签页信息
- storage: 保存您的设置
- notifications: 显示复制成功通知
- clipboardWrite: 写入剪贴板
- contextMenus: 添加右键菜单项

### 💡 使用技巧

1. 使用自定义模板为特定工作流创建格式
2. 为常访问的网站设置自定义参数规则
3. 使用批量复制整理多标签页研究内容
4. 使用键盘快捷键静默复制，无需打开弹窗
5. 智能清理模式在移除跟踪参数的同时保留功能参数
